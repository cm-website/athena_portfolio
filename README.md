# Athena Bi - Personal Portfolio Website

A modern, responsive portfolio website showcasing Athena Bi's achievements in programming, music, dance, arts, and academics.

## Features

- **Modern Design**: Gradient backgrounds with floating animations
- **Responsive Layout**: Works perfectly on desktop, tablet, and mobile
- **Interactive Elements**: Smooth scrolling, hover effects, and animations
- **Comprehensive Sections**: Hero, About, Projects, Experience, Achievements, Activities, Contact
- **Professional Styling**: Purple/blue gradient theme with glass morphism effects

## Setup Instructions

1. **Add Images**: Place the following images in the `assets/` folder:
   - `athena-profile.jpg` - Professional headshot for hero section
   - `art-therapy-app.jpg` - Screenshot of the art therapy platform
   - `abandoned-village.jpg` - Screenshot of the mobile game
   - `wattpad-books.jpg` - Image representing published books
   - `reading.jpg` - Photo of Athena reading
   - `dance.jpg` - Dance performance photo
   - `violin.jpg` - Photo playing violin
   - `charity.jpg` - Charity exhibition photo
   - `programming.jpg` - Photo coding/programming
   - `writing.jpg` - Photo writing or with books

2. **Update Contact Information**:
   - Replace `athena.bi@email.com` with actual email in `index.html`
   - Update any other contact details as needed

3. **Customize Content**:
   - Review all sections and update with specific details
   - Add actual project links in the projects section
   - Update social media links in the footer
